# Scene Setup

Setting up a new scene with COZY is incredibly easy! Whether you're a seasoned developer or just starting your journey, setting up your scene is a crucial first step in bringing your vision to life.

{% hint style="info" %}
Be sure to [setup your project](https://distant-lands.gitbook.io/cozy-stylized-weather-documentation/getting-started/setting-up-your-project) before setting up a scene! It will make this process much easier and look better out of the box.
{% endhint %}

{% stepper %}
{% step %}
### Open your Scene
{% endstep %}

{% step %}
### Run the Setup Command

<figure><img src="../.gitbook/assets/image (14).png" alt="" width="548"><figcaption></figcaption></figure>

Navigate to Tools/COZY: Weather 4/Setup Scene

If you want a basic setup ready to go out of the box, use the Basic Module option. For a clean setup that starts with no modules, use the No Modules option.
{% endstep %}

{% step %}
### Press Play

Your scene is now setup for use with COZY 4!
{% endstep %}
{% endstepper %}

For the best results, setup a [track-point.md](../using-cozy/utilities/track-point.md "mention") if you do not want COZY to follow your main camera.
