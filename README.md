# Welcome

{% columns %}
{% column valign="middle" %}
### Make Your World COZY

COZY: Stylized Weather is a complete, modular environmental system designed to give you full control over the skies, weather, and time in your Unity projects. COZY provides the tools to craft dynamic, visually distinct worlds without compromising on performance.
{% endcolumn %}

{% column valign="middle" %}
{% embed url="https://www.youtube.com/watch?v=uY1uGIZfGZ0" %}
{% endcolumn %}
{% endcolumns %}

Choosing the right assets and architecture for your game is a significant decision. I want to extend a sincere thank you for considering COZY for your project; I am genuinely grateful that you would trust this system to help bring your vision to life.

This documentation is built to support you from your first installation through final optimization. It covers the core concepts of the weather and time systems, detailed breakdowns of individual modules like clouds and fog, and technical references for utilizing the C# API and integrating with your own custom logic.
